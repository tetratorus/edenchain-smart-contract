# EdenChain Smart Contracts

**ERC20**

Decimal factor: 18
Name: Eden Coin
Symbol: EDEN

**Holding contracts**

- 5 months lockup
- 1 year lockup

**Deployment scripts**
- Mass issuance of tokens
- Issuance of locked-up tokens


For linting, install solium:
```
npm install -g solium
```
then run:
```
npm run lint
```
